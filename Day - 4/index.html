List in HTML

List are used to represent real life list data.

Unordered List

ul tag: it is used to create an unordered list. The items in the list are marked with bullets.
e.g. shopping list, to-do list, etc.
list tag: it is used to create a list item. It is used inside the ul tag.
  • Mango
  • Banana
  • Apple
  • Jackfruit
  • grapes

Ordered List

ol tag: it is used to create an ordered list. The items in the list are marked with numbers.
e.g. steps to do something, etc.
list tag: it is used to create a list item. It is used inside the ol tag.
  1. Step 1
  2. Step 2
  3. Step 3
  4. Step 4
  5. Step 5
