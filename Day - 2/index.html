br tag: used to create a line break in the text.
tag: used to create a horizontal line.

p tag: used to define a paragraph of text.

bold tag: used to make text bold.Bold Text
italic tag: used to make text italic.Italic Text
underline tag: used to underline text.
strong tag: used to define important text.
em tag: used to emphasize text.
mark tag: used to highlight text.
small tag: used to define smaller text.
del tag: used to define deleted text.
ins tag: used to define inserted text.
sub tag: used to define subscript text.
e.g

H2O => Water

sup tag: used to define superscript text.
e.g

E = mc2

code tag: used to define a piece of computer code.
pre tag: used to define preformatted text.

blockquote tag: used to define a section that is quoted from another source.

address tag: used to define contact information for the author/owner of a document.

div tag: used to define a division or section in an HTML document.

span tag: used to group inline-elements in a document.
